   Задание 1

Предварительно нужно создать папку в которую мы хотим скопировать репозиторий.
Зайдя в проект, который мы будем копировать, нажимает кнопку Code.
В открытом окошке проверяем что подчеркивание находится под SSH и копируем ссылку.
В консоли переходим в котолог где будем хранить копию репозитория.
Пишем команду git clone ссылка, которую мы скопировали и нажимаем ENTER.

  Задание 2

Перейти в каталог проекта cd /путь к каталогу/
Выполнить команду git init
Далее git add . "." - все файлы в каталоге. При добавлении файла генерируется хеш значение.
git commit -m "Комментарий"

Для отправки репозиторя нужно воспользоваться командой 
git remote add имя git@github.com:логин/название репозитория.git
Отправить git push -u что бы установить отслеживание.
При последующих отправках достаточно команды git push

  Задание 3

При помощи команды cd перейти в каталог
Ввести команду git pull

  Задание 4

git status - просмотр текущего состояния
git log - просмотр списка всех коммитов
git remote -v - просмотр результата добавления репозитория
